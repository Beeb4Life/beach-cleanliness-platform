# Samudra Seva - Beach Cleanup Coordination Platform

A React-based web application that enables real-time beach cleanliness reporting, volunteer coordination, and NGO-led cleanup campaign management for Mumbai's coastal communities.

## Description

Samudra Seva (Ocean Service) is a smart web-based system designed to address the growing challenge of beach waste accumulation in Mumbai. The platform provides:

- **Real-time Beach Status Reporting** - Track cleanliness levels at Juhu Beach, Versova Beach, Girgaon Chowpatty, and other Mumbai beaches
- **Volunteer Coordination** - Connect volunteers with cleanup drives, track participation, and recognize contributions
- **NGO Planning & Analytics** - Help environmental organizations identify priority locations and measure their impact
- **Community Engagement** - Foster awareness and participation in coastal conservation efforts

This platform bridges the gap between beach visitors, volunteers, and environmental organizations to ensure coordinated and effective cleanup efforts.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
    ```
    git clone <repository-url>
    ```

2. Navigate to the project directory:
    ```
    cd samudra-seva/reactjs
    ```

3. Install dependencies:
    ```
    npm install
    ```

4. Start the development server:
    ```
    npm run dev
    ```
